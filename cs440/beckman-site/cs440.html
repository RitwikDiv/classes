Welcome to CS 440!

This class, Programming Languages and Translators, will teach you what programming languages are and how they are made. Most programmers use the language they use simply because it’s the only language they know. When you are done with this course, you will be able to select the right programming language for the task at hand, and you will be able to implement an interpreter for it if an implementation is not conveniently avaialable.
