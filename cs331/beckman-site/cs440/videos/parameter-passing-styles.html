Parameter Passing Styles

Objectives

  • Explain the difference between strict and non-strict parameter passing styles.
  • Understand the effect of these strict parameter passing styles:
    • Call by value
    • Call by reference
    • Call by result
  • Understand the difference between these non-strict parameter passing styles.
    • Call by name
    • Call by need
  • Explain how to implement call by need using state.
